Gmail AI Agent

“From inbox to Slack, minus the noise.”

About This Project

This project combines two connected workflows to streamline how incoming enquiries and emails are handled.

The first workflow captures form submissions, classifies them with AI, and directs them into the right path - either scheduling follow-ups or gracefully declining irrelevant entries. The second workflow then listens to Gmail for incoming enquiries, automatically labels and summarizes the content, and prepares a draft response. Instead of sending replies directly, the draft is pushed to Slack, where a human teammate can quickly review, edit if needed, and approve before the message goes out.

Together, these workflows reduce manual sorting, ensure every enquiry gets acknowledged, and give teams a faster way to collaborate on responses without sacrificing quality or human oversight.

Tech Stack

  • n8n
  • OpenAI Chat Model
  • Gmail
  • Slack
  • Structured Output Parser
