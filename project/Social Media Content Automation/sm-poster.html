About This Workflow

This project automates the process of creating and publishing social media posts across multiple platforms. It fetches content ideas from RSS feeds, AI prompts, or uploaded files, then formats them according to each platform’s style guide. Images and captions are automatically generated or enhanced using AI. Posts are scheduled at optimal times for maximum engagement. This workflow eliminates repetitive manual posting, freeing up time for strategic content planning. It’s scalable and can be adapted for individuals, agencies, or large brands.

Tech Used

  • n8n
  • RSS Feed Trigger Node
  • HTTP Request Node (for API posting)
  • OAuth2 Authentication (Twitter/X, LinkedIn, etc.)
